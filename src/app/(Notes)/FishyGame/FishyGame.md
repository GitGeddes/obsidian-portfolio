# Fishy Game
A simple game inspired by the classic Fishy Flash game. Created to learn how to make a simple game in Unity.
####
**Source code:** [FishyGame](https://github.com/GitGeddes/FishyGame)

**Download page:** [GitHub Releases](https://github.com/GitGeddes/FishyGame/releases)
####
Made with the [Unity](https://unity.com/) game engine.

---
## Gameplay
- Eat smaller fish for points and size, avoid larger fish
- Win and lose conditions
- Pause menu
- Basic settings menu

## Controls
- WASD: movement

## Settings
- Fullscreen toggle
- Various resolution options
